# Photo Organization Roadmap

## Phase 34

Asteria now has a local-first organizer foundation for mixed photo and video libraries:

- media library concept for photos, videos, and folders
- lightweight organization metadata per asset
- organizer overview and suggestion cards
- smart collections for photos, videos, screenshots, unorganized media, missing metadata, large files, and duplicate placeholders
- preview-only file organization plans
- placeholder panels for people, places, and events

## Not In This Phase

- no real face recognition
- no biometric identification
- no CLIP, OCR, or embeddings
- no cloud uploads
- no destructive duplicate cleanup
- no heavy video analysis
- no real file moving without a later confirmation flow

## Phase 35

Asteria now adds Smart Folders and the first Material Vault foundation:

- smart folder detection for local folders
- metadata-only PBR material folder detection
- Material Vault view
- material-oriented smart collections and search tokens
- no 3D shader or destructive file operations yet
